기본 웹 문서의 흐름

html 태그는 크게 인라인요소와 블록 요소로 구분이 되어진다.

블록(block) 의 특징 : 한줄에 하나씩 쓰여지며, 가로 사이즈는 상위요소의 100%를 가지게 된다.
가로사이즈가 100%이기 때문에 무조건 수직(세로) 배치가 된다.
※ css상의 width, height 속성을 사용해서 사이즈 조절을 할 수 있다.
※ 세로사이즈의 경우 입력하지 않으면 height:auto를 가지게 됨.(가지고 있는 텍스트의 크기나 이미지의 크기만큼 세로사이즈가 가변적으로 변하게 됨.)
※ 블록요소의 흐름은 세로(y축)가 기준이 된다.

인라인(inline) 의 특징 : 한줄에 여러개 쓸 수가 있으며(가로,수평기준) 인라인 요소의 크기는 요소안에 쓰여진 텍스트 및 이미지의 크기까지만 사용할 수 있다.
가로사이즈의 경우 텍스트의 까지이고 세로 사이즈의 경우 텍스트의 크기와 line-height에 따라 달라진다.

(가로 세로 사이즈는 위에 언급한 것들로 지정이 되어지며 CSS의 width, height 속성으로 사이즈를 제어할 수 없다.(이미지는 예외))
※ 인라인 요소의 흐름은 가로(x축)가 기준이 된다.
