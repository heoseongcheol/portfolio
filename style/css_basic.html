내부 스타일 시트

head 태그 내부에 style 태그 안에 기술하는 것으로 스타일 시트가 비교적 짧은 경우 사용함.

외부 스타일 시트

~.CSS 파일이 별도로 생성되어지고, 사이트 규모가 큰 경우 외부스타일 시트로 사용한다(css가 길어지거나 파일이 큰 경우)
대부분 웹문서는 외부 스타일 시트로 사용하게 됨.

인라인 스타일 시트

태그 안에 style 속성을 부여후 속성값 란에 css를 기술하는 방법으로,
css 중에 우선순위가 높을 경우 사용함.
* 일반적으로 기술하는 css 중에서는 우선순위가 가장 높으나 키워드(important)를 사용하면 해당 우선순위는 무시 되기도 함.
