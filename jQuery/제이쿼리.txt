제이쿼리에서의 함수

    function 함수명(인수,인수){
        .. 해당 함수가 할일..
    }
    ==================> js 기본함수 생성법

    var 함수명 function(){
        ...
    }
    ==================> 함수명 앞쪽에서 써서 생성하는 방법

    ※ 메소드란
     - 특정 동작이나 기능을 하는 함수
     - 메소드명() : 인수가 없는 메소드
     - 메소드명(a,b) : 인수가 있는 메소드

    ※ 메소드 체이닝 (Method Chaining)
     - 여러 메소드를 연결 해서 사용 하는것.
     - "가장 첫번째에 작성한 선택요소가 현재의 선택요소가 아닐 수 있다."
        ex) #test>li>span.list

* show(), hide()
 - 기본값 : 400 (0.4초)
 - 매개변수에 숫자(정수)를 입력하면 ms 1'1000
 - 1초 동안 실행되게 하려면 1000을 입력 해야함.
 - 매개변수에 문자열을 쓸 수도 있는데 "slow", "fast"
 - 동작시 실행할 특정 무명함수 등도 사용 할 수 있다(콜백)

* css() : css 속성 제어 가능 메소드
 - 선택요소.css("프로퍼티","프로퍼티값"); 
 - 선택요소.css({
    "프로퍼티1" : "프로퍼티1값",
    "프로퍼티2" : "프로퍼티2값",
    "프로퍼티n" : "프로퍼티n값"
 }); 
 - css() 메소드로 css를 적용 시키면, 인라인 스타일 시트로 적용이 되어 우선순위가 높아지게됨.

* addClass(), removeClass(), toggleClass()
 - 선택요소.addClass() : 선택요소에 클래스명에 해당하는 클래스를 추가함
 - 선택요소.removeClass() : 선택요소에 클래스명에 해당하는 클래스를 삭제함
 - 선택요소.toggleClass() : 선택요소에 클래스명에 해당하는 클래스를 추가/삭제함

* 마우스 제어 관련 메소드
 - mouseenter() : 마우스 커서가 선택요소 안으로 들어 왔을 때
 - mouseleave() : 마우스 커서가 선택요소 밖으로 나갈 때
 - mouseout() : 마우스 커서가 선택요소 밖으로 나갈 때
 - mousedown() : 선택요소를 클릭하고 있을때 (마우스를 누르고 있을때)
 - mouseup() : 선택요소를 클릭하고 있다가 뗐을때
 - mouseover() : 마우스커서를 선택요소 안에 올렸을때

* 텍스트 변경 메소드 - text() ☆ 일반 텍스트 추가
 요소.text(문자열 or 변수명) : 해당 요소에 인수값을 넣어줌

    $(#cnt).text("텍스트");
     => cnt라는 아이디를 가진 요소 안에  '텍스트' 라는 문자열을 넣음. (기존에 있던 텍스트를 사라짐)

* 요소의 세로크기 관련 메소드
 slideDown(문자열 or 숫자) : 해당 요소의 세로 크기 만큼 아래로 커지면서 보여짐
                (height:0 -> height: 지정된 크기까지)
 slideUP() : 해당 요소의 세로 쿠기 만큼 위로 작아지면서 감춰 줌
                (height: 지정크기 -> height:0)
 slideToggle() : slideDown(), slideUP() 번갈아 가면서 실핼

    >> 매개변수로 숫자(정수 ms) , "slow", "fast"

* 선택요소의 자식요소 혹은 특정 자식요소를 선택 : children()
 - children() : 인수값 없이 사용 시 선택요소의 모든 자식요소를 선택
 - children("문자열") : 인수값에 문자열이 들어가게 되면 선택요소의 자식요소중
                        입력한 문자열에 해당하는 선택자만 선택

* 선택요소의 부모, 조상 요소 선택
 - parent() : 선택요소의 부모요소 선택
 - parents("선택자") : 선택요소의 조상 중에 특정 요소 선택

* $(this) : js의 this 키워드와 같음
 -> 선택자 나 자신

* index() : 제이쿼리에서 선택요소의 인덱스값을 리턴 해주는 메소드

* html() : 문자열 안에 html 태그로 구성해서 입력.
    >> text() : 문자열만 넣어줌.(html xx)

* 속성제어 메소드 - attr() :
 - attr(속성명) : 선택요소의 속성값을 가져옴
 - attr(속성명,바꿀속성값) : 선택요소의 속성명의 속성값을 바꿀 속성값으로 변경

    <img src="./img/01.png" alt="" id="test" />
     실행 >> $("#test").attr("src","./img/test.jpg");
    실행결과 >>>> <img src="./img/test.jpg" alt="" id="test" />


* 순서찾기
 - eq(index) : 선택요소의 index(정수) 값을 선택
 - .next() : 선택한 요소의 다음번째 요소 선택
 - .prev() : 선택한 요소의 이전 요소 선택
 - .nextAll() : 선택한 요소의 다음부터 모든 요소 선택(css선택자 ~ 랑 비슷)
 - .prevAll() : 선택한 요소의 이전부터 모든 요소 선택
 - .sibilings("선택자") : 형제요소 중의 특정 선택자 선택
    .sibilings() : 형제요소 선택

* 크기 리턴
 - width() : 선택요소의 가로 사이즈 리턴
 - height() : 선택요소의 세로 사이즈 리턴

* animate() 메소드
 - animate({css변경값},100}) 
  : css변경값을, 0.1초 동안 변경함. 
 - animate({css변경값},100,function(){...}) 
  : 마지막인수 해당 anumate()메소드 종료 후 실행할 실행문 

* change() : 선택요소의 값이 변경이 되거나 상태가 변할때
* find(선택자) : 선택자를 찾아 오는 메소드
* attr() : 속성값을 제어가는 메소드
* prop() : 속성값을 제어가는 메소드
 - 차이점 : attr()은 html의 속성값을 사용(문자열로 리턴)
            porp()은 자바스크립트의 속성을 취급(boolean,데이터,함수 등으로 리턴)

* effect 메소드
 - fadeOut() : 점점 사라지는 것 (opacity : 0)
 - fadeIn() : 점점 보여주는 것 (opacity : 1)
 - fadeTo(시간,투명도) : 시간 (정수) ms
                         0 ~ 1사이의 소수점값
                         (투명도 0 = 안보임, 투명도 1 = 보임)

*is() 메소드
 - is("값") : 값을 판별해서 일하는 요소를 검사 해주는 메소드(리턴값은
 boolean : 일치하면 true, 아니면 false)

*javascript의 타이머 함수*
 - setInterval(함수,밀리초) : 밀리초 간격을 기준으로 반복해서 함수 실행
 - clearInterval(함수) : setInterval로 실행붕인 함수를 중단 시킬때 사용.
 - setTimeout(함수,밀리초) : 밀리초 이후에 작업을 딱 1번 실핼
 - clearTimeout(함수) : setTimeout으로 실행중인 함수를 중단 

* clone() : 선택요소 복제
* first() : 가장 처음에 등장하는 요소(nth-child(1)과 비슷)
* last() : 가장 마지막에 등장하는 요소
* prepend("타겟") : 타겟을 선택요소의 가장 첫번째 요소로 붙여넣기
* append("타겟") : 타겟을 선택요소의 가장 마지막 요소로 붙여넣기
* remove() : 선택요소 제거(삭제)
